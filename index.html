Passport Scanner with ESP32

This project is a project in collaboration with airlines. The passport scanner will encode MRZ which contains information of a given citizen, including the type of passport, passport ID, issuing country, name, nationality, expiration date, etc. This method uses basic image processing techniques such as thresholding, morphological operations, and contour properties.
